Upload the contents of this folder to the website root.

- `index.html`
- `about.html`
- `solutions.html`
- `services.html`
- `consulting.html`
- `training.html`
- `resources.html`

- `assets/js/site.js`

## Replaces old filename issues

- Replace old `about.txt` with `about.html`.
- Replace old `Services.html` with lowercase `services.html`.
- Logo on every page links to `index.html`.
- Navigation is consistent on every page.

## Media placeholders

Each page contains one or more `.media-slot` sections. Replace those blocks with uploaded images, videos, product screenshots, team photos, resource previews, or iframe/video embeds when available.

## Form integration

Forms are currently frontend-ready prototypes. They prevent default submission and show a confirmation note. Connect them later to your preferred backend:

- Email service
- CRM
- WhatsApp flow
- OTP provider
- Calendar booking
- Google Forms / Formspree / Airtable

The Solutions form includes UI fields for company email, OTP, WhatsApp, designation, implementation timeline and optional demo slot.
